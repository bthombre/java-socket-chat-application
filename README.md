# java-socket-chat-application
Mini project: Chat application using Java Socket Programming

# Project Description

This is a mini project developed using Java Socket Programming.
The application allows multiple clients to connect to a server and exchange messages in real time.

# Features

* Client – Server communication
* Real-time messaging
* Uses Java Sockets and Streams

# Technologies Used

* Java
* Socket Programming
* Java IO Streams

# How to Run

1. Compile files
javac ChatProjectServer.java
javac ChatProjectClient.java

2. Run Server
java ChatProjectServer

3. Run Client (in another terminal)
java ChatProjectClient
